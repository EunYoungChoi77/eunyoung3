# 최은영 포트폴리오 — 애니메이션 반복 버전

- 텍스트 등장, 형광펜, Contact 밑줄 등 스크롤 등장 효과가 화면 밖으로 나갔다가 다시 들어오면 재생됩니다.
- Hero의 빛 반사 효과는 화면에 보이는 동안 약 5초 간격으로 은은하게 반복됩니다.
- 형광펜은 섹션 재진입 시 다시 그어지고, 보이는 동안에는 밝기가 미세하게 움직입니다.
- 스크롤 스냅은 사용하지 않아 자연스럽게 끝까지 스크롤됩니다.
- `prefers-reduced-motion` 설정 사용자는 애니메이션이 자동으로 최소화됩니다.

압축을 풀고 안쪽 파일 전체를 GitHub 저장소에 업로드한 뒤 Vercel에서 재배포하세요.
