# 최은영 UX/UI 포트폴리오 — Full Screen Update

## 변경 내용
- About / Approach / Contact를 브라우저 화면 높이에 맞춘 풀스크린 섹션으로 조정
- 프로젝트 카드 1개가 한 화면에 안정적으로 들어오도록 높이와 16:9 이미지 비율 조정
- 프로젝트 이미지를 HTML·JavaScript 안에 직접 포함해 Vercel에서 assets 폴더가 누락되어도 이미지가 표시되도록 수정
- 기존 레이아웃, 컬러, 타이포, 애니메이션 유지

## 배포
이 폴더의 모든 파일을 GitHub 저장소 최상위에 올린 뒤 Vercel에서 재배포하세요.
이미지는 코드에 포함되어 있어 `assets/images`가 빠져도 표시되지만, 원본 보관을 위해 폴더도 함께 두는 것을 권장합니다.

## 스크롤 오류 수정
- Safari/트랙패드에서 화면이 방방 튀던 `scroll-snap` 기능 제거
- 각 섹션은 계속 화면 높이를 채우지만 일반 웹페이지처럼 부드럽게 스크롤
- Contact 아래 이메일과 검은색 푸터까지 끝까지 내려가도록 수정
- 스크롤 중 브라우저 높이 재계산으로 프로젝트 카드가 흔들리는 현상 완화
